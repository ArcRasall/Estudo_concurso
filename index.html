Fase 2: Pós-Edital

Assim que o primeiro edital (seja Proderj, Degase ou TJ-RJ) for publicado, este cronograma deve ser adaptado.

  • Pause as matérias que não caíram no edital.
  • Adicione as matérias específicas (ex: Programação Java para o Proderj, ou Legislação Específica do TJ-RJ).
  • Foque 100% na banca organizadora do edital publicado.
